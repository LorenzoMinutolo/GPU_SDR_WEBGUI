Mearue explore help

This section contains information about the measure explorer tool. This tool is intended as a first approach to explore the measures. Many things can go wrong (i.e. the measure is not indexed, the web interface is wrong) when one attempt to use this section with custom measures/analysis/plotting. There are at least two cases that led you to this page:
  • I did everything right and the system does not work. --> read the FAQ.
  • I want-to-try/tried something new and it doesn't work. --> Scroll down and read the developmend guide.
    • FAQ
    Any restriction on filename?

    There are restriction on the filename: cannot be longer than 2048 characters and cannot contain spaces. The interface only recognises files ending with the ".h5" extension.

    How does the folder structure works?

    The main mesure folder is the one the by the path variable (you can change it in settings and explore folder section). in this Folder one can keep both: subfolders and h5 files. The interface allow to explore only the first layer of subfolders (i.e. the subfolders of subfolders will not be considered)

    Can I analyze and plot measures not taken using this inteface?

    Yes. The files have to be in the path folder or in a subdirectory. Note that many attributes (Temperatures, Other instruments values...) may not be present in the measure when using a custom script.

    Can I modidy an attribute of the measure?

    No, not using this system. In order to modify the measures attributes you have to do it manually. The easyest way is to install h5py python module and use the low level API.
